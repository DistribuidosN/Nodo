# Documentacion

Subcarpetas principales:

- `api/postman/`: colecciones y entornos para pruebas manuales.
- `demo/`: artefactos generados por la demo extremo a extremo.
- `reference/`: mapas y rutas utiles del repo.
- `reports/`: PDFs y fuentes `.tex` de informes y guias.

La raiz de `docs/` debe quedar solo para esta guia o archivos realmente
compartidos entre varias categorias.
