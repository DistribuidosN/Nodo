# Examples

Scripts manuales para probar clientes y flujos desde fuera del worker:

- `submit_task.py`: usa el contrato de control del nodo.
- `send_real_image.py`: usa el contrato de negocio con una imagen real.
- `imagenode_client.py`: cliente base para pruebas del servicio de imagen.
- `load_submitter.py`: envio repetido de carga.
- `mock_app_server.py`: callback server de prueba para progreso, resultados y heartbeat.
